# Редактор научной программы конференции

Онлайн-редактор для создания и редактирования научных программ конференций с возможностью экспорта в PDF.

## Описание

Веб-приложение для **Профессионального общества трихологов**, позволяющее редакторам создавать программы конференций в режиме онлайн и экспортировать их в формат PDF.

## Возможности

- ✅ Визуальный редактор программы конференции
- ✅ Добавление дней конференции
- ✅ Создание научных сессий
- ✅ Добавление докладов с указанием авторов
- ✅ Вставка перерывов (кофе-брейки, обеды)
- ✅ Редактирование заголовка программы
- ✅ Сохранение программы в JSON
- ✅ Загрузка ранее сохранённых программ
- ✅ **Экспорт в PDF** с сохранением форматирования
- ✅ Адаптивный дизайн
- ✅ Корпоративный стиль Профессионального общества трихологов

## Установка

### Локально

1. Склонируйте репозиторий:
```bash
git clone https://github.com/kbgdov-dev/scientific-program.git
cd scientific-program
```

2. Откройте `index.html` в браузере

Никаких дополнительных зависимостей или сервера не требуется!

### На сервере

Просто загрузите все файлы на ваш веб-сервер через FTP/SFTP.

## Использование

### Создание новой программы

1. Откройте редактор в браузере
2. Нажмите **"Редактировать заголовок"** для изменения названия конференции, подзаголовка и даты
3. Используйте кнопки на панели инструментов:
   - **Добавить день** - создать новый день конференции
   - **Добавить сессию** - добавить научную сессию с указанием времени и названия
   - **Добавить доклад** - добавить доклад в последнюю созданную сессию
   - **Добавить перерыв** - вставить кофе-брейк или обед

4. Кликните на любой элемент программы для его редактирования

### Сохранение и загрузка

- **Сохранить** - сохраняет текущую программу в формате JSON
- **Загрузить** - загружает ранее сохранённую программу из JSON-файла

### Экспорт в PDF

1. Нажмите кнопку **"Скачать PDF"**
2. Дождитесь генерации PDF (несколько секунд)
3. Файл автоматически сохранится на ваш компьютер

## Настройка

### Изменение корпоративных цветов

Откройте `assets/css/style.css` и измените CSS-переменные:

```css
:root {
    --primary-color: #E8B4D9;    /* Основной розовый цвет */
    --secondary-color: #8B7D82;  /* Серый цвет для текста */
    --text-color: #333;          /* Цвет основного текста */
}
```

### Изменение логотипа

Замените файл `assets/images/logo-t.png` на свой логотип.
